# 🌸 SilkWax Studio

> *Smooth. Confident. You.*

A modern, mobile-friendly website for SilkWax Studio — a premium waxing & hair removal salon.

## Features
- Hero section with brand identity
- Services listing with pricing
- About Us section
- Customer testimonials
- Appointment booking / registration form
- Responsive mobile-first design

## Tech Stack
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
